# 10. 데크, 우선순위 큐

- 데크 : 스택과 큐의 연산을 모두 갖고 있는 복합자료형
- 우선순위 큐 : 추출 순서가 일정하게 정해져 있지 않은 큐

### 데크

- `Double-Ended Queue`, 양쪽 끝을 모두 O(1)에 추출할 수 있는, 큐를 일반화한 형태의 추상자료형 (ADT)
- 배열로도 원형큐와 같이 구현이 가능하지만, 이중연결리스트로 구현하는 것이 잘 어울림.
  - See [26\_원형데크디자인.py](26_원형데크디자인.py)
